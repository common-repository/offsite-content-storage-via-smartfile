== Description ==

The Smartfile external storage plugin allows you to store your media files on Smartfile and serve these files from Smartfile services, this would help you save space on your hosting and reduce the load on your site. Think of it like a CDN (not really) that offloads files on your site to Smartfiles own servers.

== Installation ==

Installing "Offsite Content Storage via Smartfile" can be done either by searching for "Smartfile Storage for Wordpress" via the "Plugins > Add New" screen in your WordPress dashboard, or by using the following steps:

1. Download the plugin via WordPress.org
1. Upload the ZIP file through the 'Plugins > Add New > Upload' screen in your WordPress dashboard
1. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= Can I still keep copies of media files on my site, in case smartfile goes down? =

Yes, you can; there's an option for this in the Smartfile plugin settings page.
